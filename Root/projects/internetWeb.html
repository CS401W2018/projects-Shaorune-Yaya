1.Internet

Internet is a network that connected devices around the world, allowing them to exchange data or communicate

2.Web

Web is a system that access through internet, it allows user to visit or browse pages through their device.

3.How are the Internet and Web different?

Internet is the infrastructure of networks around word.
Web is the system that run on the internet.

4.ASCII and Unicode

ASCII is a earlier Character encoding standard, mainly uses for English.

Unicode is a world wide language encoding standard, it supports over 140,000 characters and will support every language around the world.

5.What is UTF-8

UTF-8 is used for encoding Unicode, it will use 1 - 4 byte for different characters. UTF-8 will also be compatible with ASCII.

6.URL

URL are used for locate the addresses of the resource on the internet, by using it we can visit website, files, or other internet services.

7.Domain Name

Domain name is a easier name for identify website or other sources. Each domain name will targeting a IP address.

8.HTTP

HTTP is a protocol that uses on Web, it defined the communication between browser and server.

9.HTTPS

HTTPS is a safer verson of HTTP, it uses SSL/TLS to secure the data, make sure data between browser and server are encrypted during transmission.

10.IP Address

IP Address is a Unique address for end devices. It has IPv4 or IPv6, they both ensure the communication between devices

11.DNS

DNS, Domain Name System, is a system that convert Domain Name to IP Address that allow end devices to access.

12.W3C Recommendations

W3C is a standard that published by W3C, it covers the aspect of website, style, security, and accessibility to ensure the consistency between different devices
