Four ways to select tags

  1. Tag Name
  2. Decendent
  3. ID(Only uses once, noun)
    a tag can only have one ID
    by assign id to tags after their tag for example <p id = "first" >
    Only one word is accepted, if multiple, use connection like _ or #
  4. Class(Re-useable, adjective)
    a tag can have multiple classes
    Works exactly like ID but start with a - or .

What is the difference between an ID and a Class?

Class name can be used multiple times, but Id can only be used one time, we should defalutly uses class rather than ID

What is the difference between a SPAN tag and a DIV tag?

SPAN: is a inline element, its use for covering a part of the paragraph so we cna edit CSS without linebreaks
DIV: is a block element, its use for covering a larger block or multiple tags, DIV tags causes linebreaks.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

There is not much different between them, except DIV doesn't define the tag characteristic. and tags like HEADER etc can define the usage of the block
