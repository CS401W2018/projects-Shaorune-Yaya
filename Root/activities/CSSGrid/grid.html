  3. What is the main difference between CSS Grid and CSS Flex?

    4. Grid is two dimensional, Flex is only one dimensional

    Flex: Parent (container): Controls the direction, alignment, and spacing of child elements in one direction (either row or column).
    Child (item): Controls how it grows, shrinks, and aligns within the parent.

    Grid: Parent (container): Controls both rows and columns, creating a grid layout.
    Child (item): Controls how many rows or columns it spans and its alignment within the grid.
