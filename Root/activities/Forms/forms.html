Gender:

  1. The form tag has an "action" attribute. What does it do?

    2. Specify the destination URL for sending data after submitting the form, which is the address where the server receives the data.

  2. The form tag has a "method" attribute. What does it do?

    3. Define the submission method for form data, usually "GET" or "POST".

  3. What is the difference between a "POST" and a "GET"?

    4. GET: The data is attached to the URL, which is suitable for obtaining data. The amount of data is limited and not secure.
    POST: Data is sent within the request body, making it suitable for submitting sensitive or large amounts of data and more secure.
