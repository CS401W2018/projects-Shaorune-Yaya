Cards

  1. Why do we need a viewport meta tag?

    2. Meta tags are used to control the layout and scaling behavior of web pages on mobile devices. It ensures that the page displays as expected on screens of different sizes, especially on mobile devices.

  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

    3. The priority depends on the order of CSS rules, and rules defined later will override those defined earlier.

  3. How do you change the CSS for a web page displayed on a phone?

    4. use @ media (max width: 600px) {...} to apply CSS that only takes effect when the screen width is less than 600px, thereby adjusting the display effect of the page on the phone.
