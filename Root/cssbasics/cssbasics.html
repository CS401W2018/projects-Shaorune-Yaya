CSS Basics

CSS(Cascading Style Sheets) is what we use to make our website pretty
HTML Attributes are different, it can change the color, font, etc.They are old and not in used today, but our browser will still surport them but if we use, it will overide all CSS codes.

Inline CSS

Instead, we can use inline CSS, but still not recommanded, we can put inline CSS in every tag beside title

Embedded CSS

By adding style tag, we can change whole page color

External CSS

External CSS is the same within style tag, but as a seperated stylesheet file, we should use external CSS in order to unify the style of the other pages of the site

CSS selecters

Selecters are to select a specific tag for example <p> or <h1> so we can edit specificly rather than change everything
